# Employee Management System

Full-stack application to manage employee records (CRUD operations) 

## Tech Stack

**Frontend**
- React 19
- Axios (for API calls)
- CSS / Bootstrap 

**Backend**
- Java 17 
- Spring Boot 5.1
- Spring Data JPA
- Spring web
- MySQL
